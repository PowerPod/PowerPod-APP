# Powerpod.pro App 
build with ReactNative & Expo

## Framework used in the project
1. ReactNative
2. Expo

## Libraries that the project depends on
1. Particle.network
2. react-native-ui-lib
3. zustand


## Usage

1. install dependencies
```shell
npm install
```

2. install ios dependencies
```shell
pod install
```

3. run the project
```shell
expo run ios
```
